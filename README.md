STEPS TO RUN APP LOCALLY
    1. clone the repository
    2. cd to the repository (`cd riktam-tech-assignment`)
    3. run `npm install`
    4. run `npm start`

STEPS TO EXECUTE TEST CASES
    1. run `npm run test`
